HTML Fundamentals: File Paths & Structure

File Path System

File paths are essential for linking files on your website. The video explains two types: absolute paths and relative paths.

Relative vs. Absolute Paths

  • Relative Path: The path to a file relative to the current file. This is the most common type used in development.
    Example: `images/my-photo.jpg`
  • Absolute Path: The full path to a file, starting from the root directory or a complete URL.
    Example: `https://www.example.com/images/my-photo.jpg`

Div, Class, and ID

These elements are used for organizing and identifying parts of your HTML code.

  • : A generic container used to group content. It's often used with CSS to apply styles to a section of a webpage.
  • class: An attribute used to group multiple HTML elements together. You can use the same class name on multiple elements.
  • id: An attribute used to uniquely identify a single HTML element on a page. Each id must be unique.

Web Page Structure

Webpages are typically divided into three main sections to provide clarity and meaning:

  • Header: The top section of a page, often containing a title or logo.
  • Main: The primary content of the page.
  • Footer: The bottom section of a page, usually containing copyright information or contact details.
